# ForgeNote

A high-fidelity, native Jetpack Compose productivity application engineered to manage structural text notes, organize highly optimized AI prompt templates, and stream context-aware conversations directly with Google's Gemini Flash model.

## Core Features
- **Notes Vault:** Full mechanical card editor layout with canvas-driven ruled notebook paper textures and native storage exporting (.html / .md).
- **Prompt Lab:** Automated template creation workbench with tags (Code, Creative, Analysis), system clipboard matching, and direct AI piping vectors.
- **Gemini Terminal:** Session-isolated conversational environment leveraging coroutine stream token channels with tactile user/machine speech bubbles.

## Tech Stack
- **Language:** Kotlin 100%
- **UI Architecture:** Jetpack Compose + Material 3 Design Matrix
- **Data Layer:** Room Database Engine for continuous offline caching
- **Intelligence:** Google Generative AI SDK (com.google.ai.client.generativeai)
- **CI/CD Pipeline:** GitHub Actions automated binary compilation engine
